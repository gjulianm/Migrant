Migrant 0.3.3

This is the Migrant project by Ant Micro, a fast and flexible serialization framework usable for undecorated classes, written in C#. Its aim is to provide an easy way to serialize complex graphs of objects, with minimal programming effort.

Features

  • designed for ease of use -- call one method to serialize, one to deserialize an object
  • works out of the box for nested objects, value and reference types and complex collections
  • graceful failure for objects which should not be serialized (e.g. pointers)
  • compact, endianness-independent data format
  • takes into account complex graph relations between objects
  • allows custom (de)serialization patterns for specific types
  • supports Surrogates
  • (new) can use generated methods rather than reflection for both de- and serialization for increased speed
  • (extra) mechanism to translate primitive .NET types into a compact binary representation
  • (extra) deep cloning of objects
